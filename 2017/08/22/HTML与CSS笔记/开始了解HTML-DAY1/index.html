开始了解HTML-DAY1

什么是HTML

HTML是一种标记语言，一般以成对的方式来使用。

<html> 
    <head>
        <title>这是网页的首部</title>
    </head>

    <body>
        <h1>一级标题</h1>
        <h2>二级标题</h2>
        <p>内容段落</p>
    </body>
</html>

一般的，元素由：开始标记+内容+结束标记组成。

什么是CSS

CSS是层叠样式表。是用来控制HTML的表现。以形式放在里面。

<html> 
    <head>
        <title>这是网页的首部</title>
        <style type ="text/css">
            body{
                background-color:#d2b48c; 设置背景色为土黄色
                margin-left:20%;
                margin-right:20%; 设置左右外边距分别占页面的20%

                border:2px dotted black; 定义页面主体周围的边框是虚线，颜色为黑色。
                padding:10px 10px 10px 10px; 在页面主题周围创建一些内边距
                font-family:sans-serif;定义文本使用的字体
                }
        </style>
    </head>

    <body>
        <h1>一级标题</h1>
        <h2>二级标题</h2>
        <p>内容段落</p>
    </body>
</html>
